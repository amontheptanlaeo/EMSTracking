# EMS Tracking API
* เช็คสถานะพัสดุ EMS ด้วย Axios บนเว็ปแอป Reactjs
# Installation
* npm install
* ใส่ Token ของตัวเองใน headers บรรทัดที่ : 17 ไฟล์ App.js
  * ```javascript
    headers: {
      'Authorization': 'Token ...',
      'Content-Type': 'application/json'
    }
    ```
* npm start
# Get Token
